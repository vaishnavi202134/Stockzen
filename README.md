## About The Project

StockZen is a modern stock trading web application developed using the MERN stack.  
It provides users with an interactive trading dashboard, portfolio management system, watchlists, and real-time stock insights through a clean and responsive interface.

The platform is designed to simulate a real-world trading experience with secure authentication, scalable backend architecture, and smooth UI interactions.

# Installation & Setup

## 1️Clone the Repository

```bash
git clone https://github.com/your-username/StockZen.git
```

---

## 2️Navigate to Project Folder

```bash
cd StockZen
```

---

# 🔧 Backend Setup

```bash
cd backend
npm install
npm start
```

Backend runs on:

```bash
http://localhost:3030
```

---

# 💻 Frontend Setup

Open a new terminal:

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```

---

# 📊 Dashboard Setup

Open another terminal:

```bash
cd dashboard
npm install
npm start
```

Dashboard runs on:

```bash
http://localhost:3001
```

---

# 🔐 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000
MONGO_URL=your_mongodb_connection
JWT_SECRET=your_secret_key
```
